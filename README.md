# quasarTest


## 처음 진입 시

```
server 디렉토리, front 디렉토리 둘다 

npm install
```

server 디렉토리에 .env 파일 생성 (위치 server/ )

**.env
```
DB_URL=mongodb://<mongodb 계정>:<pwd>@<host>:<port>/<collection name>
```


## server 실행 명령어
```
npm start
```


## front를 실행하기 위해서는 Quasar가 설치가 되어있어야 한다.

## front 실행 명령어
```
quasar dev
```
